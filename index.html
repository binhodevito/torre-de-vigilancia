O que é a Torre de Vigilância

A Torre de Vigilância é uma ferramenta gratuita para organizar sua leitura e coleção de quadrinhos. Registre o que leu, o que está lendo e o que quer ler. Dê notas, escreva impressões, separe a coleção física do histórico de leitura e crie listas temáticas.

Feita para toda a comunidade de leitores — não importa se você lê Marvel, DC, mangá ou quadrinhos nacionais. Cada usuário tem sua própria conta, privada e só sua.

Não há fins lucrativos com este projeto e nunca haverá. Nenhuma propaganda, nenhum plano pago, nenhuma venda de dados. É uma ferramenta criada por um leitor para leitores — e assim vai continuar.

Filosofia do projeto

  • Gratuito para sempre, sem anúncios e sem fins lucrativos
  • Seus dados são seus — exportáveis a qualquer momento em JSON
  • Código aberto — disponível no GitHub para qualquer um copiar e adaptar
  • Feito por leitores para toda a comunidade de quadrinhos

Por que criei isso

Sou jornalista e estrategista de conteúdo — passo o dia pensando em como organizar informação de forma que faça sentido para as pessoas. Com quadrinhos, nunca consegui fazer o mesmo pela minha própria coleção.

Tentei planilhas, aplicativos genéricos, listas no celular. Nenhum entendia o contexto de quem coleciona: editoras, números de edição, a diferença entre ter o físico e ter lido, e principalmente comparar os próprios hábitos com os de pessoas próximas. Queria saber: o que meus amigos estão lendo? Quantos gibis eu realmente li no último ano?

Além disso, estava procurando um projeto real para aprender SQL de forma prática — não com dados inventados, mas com algo que fizesse sentido pra mim. Modelar uma coleção de quadrinhos foi o pretexto perfeito. O que começou como exercício virou ferramenta, e a ferramenta virou esse projeto aberto.

O Guia dos Quadrinhos já tem todo o catálogo que eu precisava — faltava uma ferramenta para conectar esse acervo ao controle pessoal de leitura. A Torre de Vigilância é essa ponte. Feita por um leitor, para leitores.

Créditos e agradecimentos

Todo o catálogo de edições, capas e informações que você vê nesta ferramenta vem do Guia dos Quadrinhos — um projeto mantido há décadas por fãs voluntários, com amor e dedicação à memória dos quadrinhos brasileiros. É o maior e mais completo acervo do gênero. Um patrimônio construído pela comunidade, para a comunidade.

Se você ainda não conhece o Guia, vai por mim: é um lugar incrível. Você vai perder horas descobrindo títulos que não sabia que existiam, encontrando edições raras, e mergulhando fundo na história dos quadrinhos publicados no Brasil. Visite. Explore. Contribua se puder. Eles merecem todo o crédito — e todo o reconhecimento da comunidade.

A Torre de Vigilância consulta o Guia de forma respeitosa: cada edição é acessada apenas uma vez, com delay entre requisições, e o resultado fica em cache permanente para nunca sobrecarregar os servidores deles. Os dados são deles. A ferramenta é apenas uma ponte.
